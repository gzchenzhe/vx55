# 微信聊天截图生成器

一个基于纯前端实现的微信聊天截图编辑器，用来在浏览器中编排聊天内容、预览 iPhone/微信风格界面，并导出高清 PNG 截图。

> 仅用于学习、演示、设计稿预览等正当场景。请勿用于伪造证据、冒充他人或误导传播。

## 功能特性

- 微信聊天界面实时预览，包含状态栏、导航栏、消息区和底部输入栏。
- 支持浅色模式、深色模式切换。
- 可编辑手机时间、返回应用文案、聊天标题、未读数、电量、网络标识。
- 可设置双方头像、对方昵称，并控制是否显示昵称。
- 支持多种消息类型：
  - 文字消息
  - 图片消息
  - 语音消息，含时长、未读、已读、转文字状态
  - 转账消息，含金额、副标题、底部文案、待接收/已接收状态
  - 语音通话
  - 拍一拍/系统提示
  - 时间戳
- 支持拖拽调整消息顺序。
- 自动保存编辑状态到浏览器 `localStorage`，刷新页面后可恢复。
- 使用 `html-to-image` 按 3 倍像素比例导出高清截图。

## 技术栈

- HTML / CSS / JavaScript
- Vue 3，全局构建版本，位于 `vendor/vue.global.prod.js`
- Tailwind CSS 本地样式文件，位于 `vendor/tailwind-local.css`
- html-to-image，位于 `vendor/html-to-image.min.js`
- 本地字体与图片资源，位于 `fonts/` 和 `pic/`

## 使用说明

1. 在左侧编辑面板设置全局外观、人物信息和聊天内容。
2. 点击底部按钮添加文字、图片、语音、转账、通话、拍一拍或时间戳。
3. 在消息模块中勾选“是我发出的”切换消息方向。
4. 长按并拖拽消息模块可调整聊天顺序。
5. 右侧手机预览会实时更新。
6. 点击右上角“导出截图”生成 PNG 文件。
7. 点击“重置”可清空本地保存的数据并恢复默认内容。
## 数据保存

编辑内容会自动保存到浏览器本地存储：
上传头像或图片时，图片会以 Base64 形式存入本地存储。如果图片太大，可能触发浏览器约 5MB 的存储限制。遇到保存失败时，请压缩图片后重新上传。

## 常见问题

- 导出截图失败

确认 `vendor/html-to-image.min.js` 能正常加载。建议通过本地服务或正式静态服务器访问页面，而不是在受限制的文件环境中运行。

- 字体或图标显示异常

确认 `fonts/`、`pic/`、`vendor/` 目录与 `index.html` 在同一项目根目录下，并且部署后路径没有被改动。

- 刷新后内容没有保存

通常是图片过大导致 `localStorage` 超出限制。可以压缩上传图片，或点击“重置”清空旧数据后重新编辑。
如需新增消息类型，通常需要同时补充三处：

1. 编辑面板中的消息配置表单。
2. 右侧预览区域的消息渲染模板。
3. Vue `addMessage` 方法中的默认数据。
